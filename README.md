# vue-sell

> 使用vue1.0开发的高仿饿了么点餐WebApp

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# 模拟服务器运行
node prod.server.js
```
